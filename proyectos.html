ANDROID

Android is an operating system initially intended for mobile phones. What makes it different is that it is based on Linux, a free, cross-platform, free operating system core. The system allows you to program applications in a Java variation called Dalvik. The operating system provides all the necessary interfaces to develop applications that access the telephone functions (such as GPS, calls, calendar, etc.) in a very simple way in a well-known programming language such as Java
